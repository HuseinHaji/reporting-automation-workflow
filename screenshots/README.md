# Screenshot Guide

Add these screenshots after running the Streamlit app or building a Power BI dashboard from `output/powerbi/powerbi_reporting_dashboard.csv`:

- `executive_overview.png`: KPI cards, revenue trend, exception severity overview
- `monthly_reporting_summary.png`: monthly KPI table, revenue vs target, collection and overdue trends
- `exception_report.png`: exception table plus severity/type charts
- `data_quality_overview.png`: data quality check table and failed-record view
- `customer_reporting_table.png`: customer drilldown with outstanding amount, overdue ratio, risk rating, and reporting status

Run the demo with:

```bash
streamlit run src/app.py
```
